Step-by-step

Installation
Guide

Follow the steps below for your platform. Everything runs locally — no accounts needed.

1

Download the Extension

Scroll up to the Extension section and click the Add to Chrome button. A .zip file will land in your Downloads folder.

2

Unzip the Extension

Right-click the downloaded .zip and extract it to a permanent folder on your PC — for example C:\CipherExtension. Don't delete this folder after installing.

3

Open Chrome Extensions

In Chrome, click the puzzle-piece icon (🧩) in the top-right toolbar, then click Manage extensions — or just navigate to chrome://extensions.

4

Enable Developer Mode & Load Unpacked

On the Extensions page, toggle Developer mode on (top-right corner). A toolbar will appear — click Load unpacked.

✓

Select the Extension Folder

Navigate to the folder you unzipped. Inside you'll see exactly two folders: icons and fonts. That means you're in the right place. Click Select Folder — Cipher will appear in your extensions list.

🎉 Cipher is now installed. Open any chess site and the overlay will appear automatically.

4

Open the Browser Menu

Tap the three dots (⋮) icon in the top-right corner of the browser.

5

Tap Settings

Select Settings from the dropdown menu.

6

Tap Extensions

Inside Settings, find and tap Extensions.

7

Open Developer Options

On the Extensions screen, tap Developer options.

8

Install from .zip

Tap "From .zip / .crx / .user.js", then navigate to the Cipher extension file you downloaded and select it. No unzipping needed — the browser handles it.

🎉 Extension installed. Open any chess site in this browser and Cipher will appear.
