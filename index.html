Cipher Chess Coach

Avoidance Doesn't
Make You Better

Every chess player has positions they dread. For me, it was the Sicilian. I'd avoid it, play something passive, never actually learn.

What I needed was to face it repeatedly with real-time feedback not post-game analysis where I'd already internalized the wrong move, but in the moment, seeing the engine's suggested line while I'm still feeling the pressure.

That's deliberate practice. That's exposure therapy.

⚠ This tool is designed for unrated and casual training games. Use during rated play violates fair play policies.

Step-by-step

Installation
Guide

Follow the steps below for your platform. Everything runs locally — no accounts needed.

1

Download the App

Visit the Download section and grab the installer for your OS — .exe for Windows, .dmg for macOS, or .AppImage for Linux.

2

Run the Installer

Windows: Double-click cipher-setup.exe. You may see a SmartScreen warning — click More info → Run anyway.
macOS: Open the .dmg, drag CiPHER to Applications.
Linux: chmod +x cipher.AppImage then run it.

✓

Install the Browser Extension

Visit the Extension section and add CiPHER to Chrome or Firefox. Open any chess site and the overlay appears automatically.

🎉 You're ready. Open a training game and CiPHER will activate.

Part 2 — Installing the Extension
4

Open the Browser Menu

Tap the three dots (⋮) icon in the top-right corner of the browser.

5

Tap Settings

Select Settings from the dropdown menu.

6

Tap Extensions

Inside Settings, find and tap Extensions.

7

Open Developer Options

On the Extensions screen, tap Developer options.

✓

Install from .zip

Tap "From .zip / .crx / .user.js", then navigate to the Cipher extension file you downloaded and select it. No unzipping needed — the browser handles it.

🎉 Extension installed. Open any chess site in this browser and Cipher will appear.
Part 3 — Installing the Android App
9

Allow Installation from Unknown Sources

Go to Settings → Security (or Privacy → Install unknown apps on newer Android) and enable installs from your file manager or browser.

✓

Install the APK

Tap the downloaded cipher.apk file to install it. Google may show a warning that the app is potentially harmful — this is expected for any sideloaded app that isn't on the Play Store.

⚠
If you see a "harmful app" prompt:
Tap the ↓ down-arrow or More details button and choose "Install anyway".
🛡
Alternative — disable Play Protect:
Open the Play Store → tap your profile icon → Play Protect → Settings → toggle Scan apps with Play Protect off. Install the APK, then re-enable it afterwards.
